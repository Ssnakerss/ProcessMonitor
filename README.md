# Parental Control — Родительский контроль для Windows

Приложение для ограничения времени использования приложений и компьютера с веб-интерфейсом управления.

## Возможности

- **Мониторинг процессов** — отслеживание запущенных приложений по имени и SHA256-хэшу
- **Ежедневные лимиты** — ограничение времени использования для каждого приложения
- **Ограничение компьютера** — общий дневной лимит времени за компьютером
- **Контроль по времени суток** — настройка разрешённых временных окон для каждого приложения
- **Блокировка по лимиту** — автоматическое завершение приложений при достижении лимита
- **Бонусное время** — возможность выдачи дополнительных минут администратором
- **Предупреждения** — уведомления о приближающемся лимите (настраиваемо)
- **Отслеживание бездействия** — пауза учёта при отсутствии активности пользователя
- **Веб-интерфейс** — админка для управления правилами, настройками и статистикой

## Требования

- Windows 7 или выше
- Go 1.25.4+

## Установка

### Вариант 1: Установка как служба Windows (рекомендуется)

```bash
# Скомпилируйте и установите службу
go build -o pc.exe ./cmd/pc
pc.exe install
```

Служба будет запущена автоматически и перезапускаться при сбоях.

### Вариант 2: Консольный режим (для отладки)

```bash
go run ./cmd/pc run
```

## Использование

### Вход в веб-интерфейс

1. Откройте браузер и перейдите на `http://localhost:8080`
2. Введите пароль: **admin** (по умолчанию)
3. После входа рекомендуется изменить пароль в настройках

### Управление правилами

- **Добавить правило** — выберите приложение, задайте лимит и временные окна
- **Редактировать** — изменить параметры существующего правила
- **Включить/выключить** — временно отключить правило без удаления
- **Удалить** — полностью удалить правило

### Настройки

- `poll_interval_sec` — интервал проверки процессов (по умолчанию: 5 сек)
- `idle_timeout_sec` — время бездействия для паузы учёта (по умолчанию: 60 сек)
- `notify_before_minutes` — за сколько минут показывать предупреждение (по умолчанию: 5)
- `computer_daily_limit_minutes` — общий лимит компьютера в минутах (0 = не ограничен)
- `web_port` — порт веб-интерфейса (по умолчанию: 8080)
- `web_bind` — IP-адрес для веб-сервера (по умолчанию: 0.0.0.0)

## Структура проекта

```
cmd/
  pc/                    # Основное приложение
    main.go              # Точка входа, установка/удаление службы
    service_windows.go   # Реализация Windows-службы

internal/
  db/                    # Работа с базой данных
    db.go                # Подключение к SQLite
    queries.go           # SQL-запросы
    migrations/          # Схемы базы данных
  models/                # Модели данных
  monitor/               # Мониторинг процессов и активности
    process.go           # Поиск и завершение процессов
    idle.go              # Отслеживание бездействия
  notifier/              # Система уведомлений
    notifier.go          # Интерфейс и базовая реализация
    windows.go           # Уведомления Windows (toast, messagebox)
  service/               # Основная логика родительского контроля
    service.go           # Цикл мониторинга и правил
    config.go            # Конфигурация
    rules.go             # Обработка временных правил
  web/                   # Веб-интерфейс
    server.go            # HTTP-сервер
    handlers.go          # Обработчики запросов
    templates/           # HTML-шаблоны
    auth.go              # Аутентификация
```

## Технические детали

- **База данных**: SQLite (хранится в `%PROGRAMDATA%\ParentalControl\data.db`)
- **Учёт времени**: секунды в базе, отображаются в минутах
- **Блокировка правил**: хранится отдельно, сбрасывается каждый день
- **Бонусное время**: выдаётся вручную, расходуется при достижении лимита
- **Уведомления**: Windows Toast (всплывающие) и Message Box (блокирующие)
- **Безопасность**: bcrypt для хранения пароля, HTTP-Only cookies для сессий

## Удаление

```bash
# Остановить и удалить службу
pc.exe uninstall

# Удалить файлы и базу данных вручную
# Файл: pc.exe
# База: %PROGRAMDATA%\ParentalControl\data.db
```
